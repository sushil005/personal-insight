# 🧠 InsightTrack - Personal Insight System

A full-stack Flask web app to track daily productivity, screen time, sleep, and mood.

## 🚀 Features
- User Authentication (Login/Register)
- Add / Edit / Delete Logs
- Data Visualization (Chart.js)
- Smart Insights
- Responsive UI (Bootstrap)

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap
- Chart.js

## 📊 Use Case
Helps users analyze productivity patterns and improve daily habits.
